serviceForUsingNamedPorts(app, ignoredLabelsopt, nameFormat) → {Service}

Generate a Service resource from either a DaemonSet, Deployment, ReplicaSet, or StatefulSet. The Service selector is derived from the .metadata.labels. All ports for all containers are exposed in the Service. Named arguments match the signature defined in (github.com/grafana/jsonnet-libs/ksonnet-util/util.libsonnet).serviceFor but use camelCase naming. This function asserts behavior that serviceFor assumes, notably that each container port is named.
Parameters:

app DaemonSet | Deployment | ReplicaSet | StatefulSet

 [] Label names that should be ignored when deriving a selector for the Service.
nameFormat string '%(container)s-%(port)s' Format for the Service port names. The format string is interpolated with an object with 'container' and 'port' members for API compatibility with the 'serviceFor' function.
